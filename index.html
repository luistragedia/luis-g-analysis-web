Qué es

Una biblioteca viva para preservar conocimiento útil.

Luis G. Analysis convierte experiencia, estudio, práctica y proyectos en conocimiento organizado, consultable y reutilizable.

Organizar

Clasificar ideas, manuales, guías, proyectos y aprendizajes para evitar que el conocimiento quede disperso.

Preservar

Guardar decisiones, procesos, errores, soluciones y buenas prácticas para que sigan siendo útiles en el futuro.

Aplicar

Transformar teoría en herramientas prácticas para proyectos, trabajo, finanzas, tecnología y vida diaria.

Documento fundador

Manual 000: Filosofía y Estándares

Manual 000 es la base editorial de Luis G. Analysis. Define la filosofía del sistema, las reglas de calidad, la estructura de categorías, los formatos oficiales y la forma en que cada libro, manual o guía deberá desarrollarse.

Antes de construir una biblioteca grande, es necesario definir cómo se organizará, cómo se escribirá y bajo qué criterios se considerará útil, clara y publicable.

Formatos

Un mismo conocimiento, varios formatos de consulta.

El contenido base podrá adaptarse a lectura web, PDF, EPUB, audiolibro, resúmenes ejecutivos y listas de verificación.

Web

Versión navegable, actualizable y organizada por categorías, capítulos y enlaces internos.

PDF

Versión fija para lectura, impresión, archivo personal y consulta estructurada.

EPUB

Versión adaptable para lectores electrónicos, móviles y tablets.

Audiolibro

Versión narrada y adaptada, no una simple lectura literal del documento.

Resumen ejecutivo

Versión breve para captar las ideas principales en pocos minutos.

Lista de verificación

Versión práctica para revisar pasos, criterios, errores y buenas prácticas.
