Glosario básico

Este glosario reúne términos básicos del Volumen I para consulta rápida. La idea es que sirva como mapa inicial antes de avanzar a temas más técnicos.

Nota de enfoque

Las definiciones están simplificadas para aprendizaje. En capítulos futuros podrán ampliarse con mayor precisión técnica.

1. Términos principales

TérminoDefinición simple
TradingComprar o vender activos buscando aprovechar movimientos de precio.
InversiónAsignación de capital con visión de mediano o largo plazo.
EspeculaciónTomar posición esperando un movimiento favorable del precio.
CapitalDinero disponible para invertir u operar.
RiesgoCantidad que se acepta perder si una operación falla.
MargenCapital bloqueado para abrir o mantener una posición apalancada.
ApalancamientoHerramienta que permite controlar una exposición mayor al capital colocado.
ExposiciónTamaño total de la posición controlada.
Stop lossSalida defensiva para limitar una pérdida.
Take profitSalida planificada para tomar beneficio.
Orden marketOrden que prioriza ejecución inmediata al precio disponible.
Orden limitOrden que define el precio máximo de compra o mínimo de venta.
TemporalidadMarco de tiempo desde el que se observa el mercado.
ScalpingOperativa de movimientos muy cortos.
IntradíaOperativa que abre y cierra posiciones dentro del mismo día.
Swing tradingOperativa de varios días o semanas.
LiquidaciónCierre forzado de una posición cuando el margen no soporta la pérdida.
DrawdownCaída del capital desde un máximo hasta un mínimo.
SetupCondición o estructura que justifica una posible operación.
InvalidaciónNivel o condición donde la idea deja de tener sentido.

2. Cómo usar este glosario

Conviene revisarlo antes de leer capítulos avanzados. Si un término no se entiende, la operación también puede entenderse mal.

3. Resumen ejecutivo

Dominar el vocabulario básico no hace rentable a un trader, pero evita confusiones peligrosas. El lenguaje claro ayuda a pensar mejor.

  • ¿Sé qué es un stop loss?
  • ¿Diferencio inversión de trading?
  • ¿Sé qué significa invalidación?
