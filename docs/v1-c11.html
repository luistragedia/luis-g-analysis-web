Checklist antes de operar

Un checklist reduce decisiones impulsivas. No garantiza ganar, pero ayuda a evitar operaciones sin plan, sin riesgo definido o bajo presión emocional.

El checklist no está para frenar oportunidades. Está para frenar errores evitables.

Nota de enfoque

Este checklist es una herramienta personal de control. Cada trader debe adaptarlo a su mercado, estrategia y nivel de experiencia.

1. Contexto

  • ¿El mercado tiene dirección clara o está lateral?
  • ¿Estoy operando contra una zona importante?
  • ¿Hay noticias o eventos relevantes?
  • ¿La temporalidad mayor apoya la idea?

2. Setup

  • ¿Existe una razón técnica para entrar?
  • ¿La entrada llega tarde o todavía tiene sentido?
  • ¿Hay confirmación suficiente?
  • ¿El precio está en una zona operativa razonable?

3. Riesgo

  • ¿Sé cuánto puedo perder?
  • ¿Tengo stop loss?
  • ¿El tamaño de posición es correcto?
  • ¿La pérdida máxima es aceptable?

4. Salida

  • ¿Tengo objetivo o zona de salida?
  • ¿Sé dónde tomar parcial si aplica?
  • ¿Sé qué haré si el precio se mueve a favor?
  • ¿Sé qué haré si se mueve en contra?

5. Estado mental

  • ¿Estoy tranquilo?
  • ¿Estoy operando por venganza?
  • ¿Estoy cansado o ansioso?
  • ¿Estoy dispuesto a aceptar la pérdida?

6. Condiciones para no operar

No operar también es una decisión. Si falta contexto, riesgo, claridad o control emocional, la mejor operación puede ser esperar.

Regla práctica: si tienes que justificar demasiado una entrada, probablemente no es tan clara.

7. Resumen ejecutivo

El checklist protege el proceso. Sirve para confirmar que la operación tiene contexto, setup, riesgo, salida y estado mental adecuados.

8. Checklist final

  • Contexto claro.
  • Setup válido.
  • Entrada razonable.
  • Stop definido.
  • Riesgo aceptable.
  • Salida planificada.
  • Mente estable.
  • Registro preparado.
