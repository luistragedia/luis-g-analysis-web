Lista de Verificación Editorial

Guía breve para revisar documentos antes de considerarlos publicables dentro de Luis G. Analysis.

1. Ortografía y acentuación

  • Revisar tildes, mayúsculas, puntuación y concordancia.
  • Evitar errores visibles en títulos, botones, subtítulos y textos destacados.
  • Leer el documento completo al menos una vez antes de publicarlo.

2. Claridad

  • Confirmar que el lector entiende qué es el tema y para qué sirve.
  • Eliminar frases innecesariamente largas.
  • Evitar jerga si no está explicada.

3. Estructura

  • Verificar títulos, subtítulos y orden lógico.
  • Confirmar que cada sección cumple una función clara.
  • Usar listas, tablas o ejemplos cuando faciliten la consulta.

4. Utilidad práctica

  • Comprobar que el documento aporta acciones, criterios o ejemplos aplicables.
  • Evitar contenido repetitivo o puramente decorativo.
  • Incluir errores comunes, buenas prácticas o lista de verificación cuando corresponda.

5. Seguridad y publicación

  • No incluir contraseñas, tokens, claves privadas ni datos personales sensibles.
  • Confirmar que los enlaces funcionan.
  • Verificar visualización en móvil antes de desplegar.
