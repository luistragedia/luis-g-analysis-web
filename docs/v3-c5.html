Order Blocks

Un Order Block es una zona donde el precio pudo haber acumulado órdenes antes de generar un desplazamiento relevante. En la práctica, se estudia como área potencial de reacción, no como garantía.

Un Order Block tiene valor cuando nace de una reacción clara y se relaciona con estructura, liquidez y desplazamiento.

Nota de enfoque

Los Order Blocks se vuelven peligrosos cuando se marcan sin criterio. La calidad de la zona importa más que el nombre.

1. Qué es un Order Block

Es una vela o zona previa a un movimiento fuerte que rompe estructura o desplaza el precio con intención. Puede actuar como referencia si el precio regresa y muestra reacción.

2. Order Block alcista

Suele ubicarse antes de un impulso alcista relevante. Puede ser una zona donde la demanda defendió precio antes de iniciar desplazamiento.

3. Order Block bajista

Suele ubicarse antes de un desplazamiento bajista fuerte. Puede representar una zona donde apareció oferta antes de romper estructura o buscar liquidez inferior.

4. Criterios de calidad

  • Debe estar ligado a desplazamiento fuerte.
  • Debe tener relación con ruptura o cambio de estructura.
  • Debe ubicarse en zona lógica.
  • Debe tener liquidez cercana o reacción previa.
  • Debe permitir invalidación clara.

5. Errores comunes

  • Marcar cualquier vela como Order Block.
  • Ignorar la temporalidad mayor.
  • Entrar sin reacción.
  • Usar zonas enormes sin gestión del riesgo.
  • Confundir zona interesante con entrada obligatoria.

6. Buenas prácticas

  • Priorizar zonas con desplazamiento claro.
  • Observar cómo vuelve el precio.
  • Combinar con FVG, liquidez y estructura.
  • Descartar zonas demasiado viejas o débiles.
  • Esperar confirmación del precio.

Resumen ejecutivo

Un Order Block es una zona potencial de reacción asociada a órdenes y desplazamiento. Su lectura debe filtrarse por estructura, liquidez, temporalidad y reacción.

Checklist

  • ¿La zona provocó desplazamiento?
  • ¿Rompió estructura?
  • ¿Tiene relación con liquidez?
  • ¿El precio volvió de forma ordenada?
  • ¿La invalidación es clara?
