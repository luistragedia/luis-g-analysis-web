Medias móviles

Las medias móviles suavizan el precio para ayudar a identificar dirección, ritmo y posibles zonas dinámicas de referencia.

Una media móvil no predice. Resume el comportamiento reciente del precio.

Nota de enfoque

Este capítulo presenta las medias móviles como herramientas de contexto dentro del análisis técnico.

1. Qué es una media móvil

Una media móvil calcula un promedio del precio durante un número de periodos. Al actualizarse con cada vela nueva, ayuda a visualizar el comportamiento general con menos ruido.

2. Tipos comunes

TipoUso básico
SMAPromedio simple, más estable.
EMADa más peso al precio reciente.
Media rápidaReacciona antes, pero genera más ruido.
Media lentaFiltra mejor, pero responde tarde.

3. Para qué sirven

  • Identificar dirección general.
  • Observar extensión del precio.
  • Comparar ritmo rápido y lento.
  • Crear referencias dinámicas.
  • Filtrar tendencia o rango.

4. Cruces de medias

Un cruce puede mostrar cambio de ritmo. En mercados laterales, los cruces suelen ser menos fiables y pueden generar ruido.

5. Medias como zonas dinámicas

En una tendencia, el precio puede reaccionar cerca de medias relevantes. La reacción del precio importa más que el simple contacto con la línea.

6. Errores comunes

  • Usar demasiadas medias.
  • Aplicarlas igual en tendencia y rango.
  • Ignorar que responden con retraso.
  • Mirarlas sin estructura.
  • Creer que una media siempre será respetada.

7. Buenas prácticas

  • Usar pocas medias.
  • Entender qué periodo representa cada una.
  • Combinarlas con estructura.
  • Evitar conclusiones en lateralidad.
  • Leerlas como apoyo, no como decisión única.

8. Resumen ejecutivo

Las medias móviles ayudan a leer dirección, ritmo y extensión. Son más útiles cuando se combinan con estructura, tendencia, zonas y volumen.

9. Lista de verificación

  • ¿La media confirma tendencia o rango?
  • ¿Estoy usando una media rápida o lenta?
  • ¿El precio está extendido?
  • ¿La lectura coincide con la estructura?
  • ¿La media aporta contexto real?
